# FrontTest

## Requirements

1. Node version 9.5.0 or greater
2. Angular CLI version 1.7.0

##How to run

1. Go to root directory and execute on terminal 'yarn'
2. Execute on terminal 'ng serve'
3. Go to the browser and enter url 'localhost:4200'
